Die main-Methode erstellt ein root-Node-Objekt und weist dem root-Knoten den Schlüsselwert 59 zu. Die früher kennengelernte Einfügen-Methode insertNodeRecursive aus der Klasse BinarySearchTree setzt ausgehend aus dem Wurzelknoten andere Werte in den binären Suchbaum nacheinander ein. Dafür wird die Instanz bst erzeugt, über welche der Zugriff auf Einfügemethode möglich.

public static void main(String[] args) {
BinarySearchTree bst = new BinarySearchTree();
Node root = new Node(59);
bst.insertNodeRecursive(root, 63);
bst.insertNodeRecursive(root, 71);
bst.insertNodeRecursive(root, 83);
bst.insertNodeRecursive(root, 92);
bst.insertNodeRecursive(root, 52);
bst.insertNodeRecursive(root, 36);
bst.insertNodeRecursive(root, 69);
bst.insertNodeRecursive(root, 48);
bst.insertNodeRecursive(root, 14);
}
