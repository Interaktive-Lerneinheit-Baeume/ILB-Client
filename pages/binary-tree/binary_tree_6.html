Die Funktionsweise des Insert-Algorithmus können Sie mihilfe dieser interaktiven Visualisierung nachvollziehen. Die Darstellung folgt dabei dem hier notierten Code-Beispiel:

public static void main(String[] args) {
BinarySearchTree bst = new BinarySearchTree();
Node root = new Node(59);
bst.insertNodeRecursive(root, 63);
bst.insertNodeRecursive(root, 71);
bst.insertNodeRecursive(root, 83);
bst.insertNodeRecursive(root, 92);
bst.insertNodeRecursive(root, 52);
bst.insertNodeRecursive(root, 36);
bst.insertNodeRecursive(root, 69);
bst.insertNodeRecursive(root, 48);
bst.insertNodeRecursive(root, 14);
}

In der main-Methode wird zuerst der leere Baum und anschließend der Wurzelknoten mit dem Wert 59 erstellt. Die zuletzt vorgestellte Methode insertNodeRecursive wird im Anschluss dazu verwendet, mehrere weitere Werte in den Suchbaum einzufügen.
