Aufgabe: Ein Suchbaum wird konstruiert

Nutzen Sie die interaktive Visualisierung, um die Konstruktion des Suchbaums schrittweise nachzuvollziehen. Beschäftigen Sie sich mit der Animation. Sie können den Ablauf selbstständig kontrollieren, indem Sie diese pausieren oder innerhalb der Wiedergabe vor- und zurückspulen.

Aufgabe: Einen Suchbaum konstruieren

Nutzen Sie diese interaktive Visualisierung, um selbstständig einen binären Suchbaum zu konstruieren. Die Visualisierung folgt dabei dem links stehenden Code. Sie können in jedem Schritt selbstständig entscheiden, an welcher Position der aktuell einzufügenden Knoten platziert werden muss. Ziehen Sie den neuen Knoten dafür auf eine der angebotenen Positionen.
