Beschäftigen Sie sich mit der Animation. Sie können die Animation selbst kontrollieren, indem Sie sie pausieren oder vorspulen können. In der Animation wird der binäre Suchbaum bst aus der Menge der einzufügenden Schlüsselwerte konstruiert. Jeder Knoten wird in den Suchbaum gemäß der Funktion insertNodeRecursive(Node head, int data) eingefügt.

Konstruieren Sie einen binären Suchbaum bst aus der Menge der schrittweise angebotenen Schlüsselwerte. Die Wurzel root des Suchbaums mit dem Schlüsselwert 59 ist schon gegeben und steht an der richtigen Position. Den weiteren Knoten mit dem Wert 63, sowie alle nachfolgenden Schlüsselwerte, sollen Sie per Drag-and-Drop selbst richtig positionieren. Jeder Knoten wird in den Suchbaum gemäß der Funktion insertNodeRecursive(Node head, int data) eingefügt.
