Für jeden Knoten k eines binären Suchbaums gilt dabei:
  • Alle Schlüsselwerte im linken Teilbaum k.left sind kleiner als der Schlüsselwert k.data.
  • Alle Schlüsselwerte im rechten Teilbaum k.right sind grösser als der Schlüsselwert k.data
Hier ist der Insert-Algorithmus für den binären Suchbaum:
public class BinarySearchTree {
private Node insertNodeRecursive(Node head, int data) {
if (head == null) {
return new Node(data);
} else if (data > head.data) {
head.right = insertNodeRecursive(head.right, data);
} else if (data <= head.data) {
head.left = insertNodeRecursive(head.left, data);
}
return head;
}
}
