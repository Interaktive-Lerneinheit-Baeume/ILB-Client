Aufgabe 7) Die Methode print wird rekursiv auf den abgebildeten binären Suchbaum angewendet. Notieren Sie die Ausgabesequenz, die durch den Aufruf der Methode erzeugt wird. Hinweis: Gehen Sie davon aus, dass die jeweiligen Knoten durch den Parameter node repräsentiert werden. Die zugehörige Klasse Node verfügt über die Eigenschaften node.data (Inhalt des Knoten), node.left (linker Kindknoten) und node.right (rechter Kindknoten). Nicht vorhandene Knoten werden durch den Wert null repräsentiert.
private void print(Node node) {
if (node == null) {
return;
}
print(node.left);
System.out.print(node.data);
print(node.right);
}
